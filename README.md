# Visual Studio Code Portfolio Website

An Visual Studio Themed Developer Portfolio built using TypeScript, Next.js and Tailwind.

Live Link: <a href="https://www.google.com/" >thierrydebelder.com</a>

> ###
>
> Ever since I started web development, I have always been fascinated with developer portfolio websites. I have found many inspiring ones, I was especially fascinated with 3D portfolio sites.
> Since I am not (yet) proficient with ThreeJS I haven't ventured into building one.
> I have, however, come across some original ones including a VScode-like clone, and I wanted to try and recreate it, and this is the result of that.
> It's not perfect and I needed to Google a lot but I got it done, and that's something I am proud of.

##

## Features

- Visual Studio Code Theme
- Projects List 
- Skills and experience section
- Working Contact page
- Misc Sections
- Styled using **Tailwind CSS**
- Written in **TypeScript**
